Для управления подачей реестров оказанной медицинской помощи используются следующие функции:

1) Загрузка файлов

2) Статус загрузки

3) Подпись файлов

Загрузка файлов

Страница загрузки файлов содержит следующую информацию:

  • Статус приема: Активно или Не активно;
  • Тип приема: Основной или Предварительный – Тип текущего приема, все реестры загруженные во время предварительного приема удаляются, при включении основного приема
  • Ваш Реестровый номер – Реестровый номер медицинской организации по справочнику F003
  • Ваша медицинская организация – наименование медицинской организации по справочнику F003
Обращаем Ваше внимание, что загрузка файлов возможна, даже если прием не активен При не активном приеме вы не сможете поставить файлы на проверку, но сможете это сделать как только прием будет включен.

К обработке принимаются:

  • файлы форматов XML или ZIP, причем во время загрузки архивы будут извлечены. Другие форматы игнорируются.
  • Файлы должны иметь правильное наименование[1];
  • Каждый файл данных должен иметь файл персональных данных;
  • Код МО в файле должен соответствовать Реестровому номеру Вашей организации.

Загрузка файлов осуществляется путем нажатие на кнопку «Загрузка файлов» или перетаскиванием файлов на нее.

Для того чтобы поставить файлы на проверку нажмите на кнопку «Отправить на обработку»

Наименование файлов

[Ti][Pi][Ni][Pp][Np]_[YY][MM][N].xml где

  • [Ti] – Тип файла
  • [Pi] - Параметр определяющий организацию – Источник(T – ТФОМС, S – СМО, M - МО);
  • [Ni] – Номер источника(реестровый номер МО);
  • [Pp]-Параметр определяющий организацию – получателя(T – ТФОМС, S – СМО, M - МО);
  • [Np] – Номер получателя;
  • [YY] – Две последние цифры порядкового номер года отчетного периода;
  • [MM] – порядковый номер месяца отчетного периода;
  • [N] – порядковый номер пакета.

Статус загрузки

Страница статуса показывает текущий активный пакет, принятый в ТФОМС Забайкальского края

Текущий пакет может быть принят как с сайта, так и с почты «VipNet»

Окно статуса загрузки содержит следующую информацию:

  • Код МО – реестровый номер медицинской организации по справочнику F003;
  • Наименование МО – наименование медицинской организации по справочнику F003;
  • Дата создания – Дата и время поступления пакета в обработку;
  • Источник – источник поступление пакета (Сайт, Почта);
  • Статус – Шаг обработки пакета(Завершено, В работе,Ошибка при проверке);
  • Комментарий – сопроводительная информация;
  • Позиция в очереди – порядковый номер в очереди на проверку.

Для получения протокола обработки нажмите на кнопку «Скачать протокол»

Данные на странице не обновляются автоматически, для актуализации информации необходимо обновить страницу или нажать на кнопку «Обновить»

Обращаем Ваше внимание на то, что для данных полученных через Сайт протокол не высылается на почту «VipNet»
